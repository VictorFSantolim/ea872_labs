Para jogar o ASCIITanks com teclado remoto, siga os seguintes passos:

1 - Abra dois terminais diferentes
2 - No terminal A, dê um make play
3 - No terminal B, dê um make runclient (esse será o terminal usado para controlar)

GitHub dessa atividade: https://github.com/VictorFSantolim/ea872_labs/tree/master/lab06
GitHub do jogo multiplayer sem teclado remoto: https://github.com/VictorFSantolim/ASCIITanks
